CSV Format

Required columns and format for the upload file

Required Columns: state, lga, ward, polling_unit, party, candidate_name, votes
Column Description Example
state State name (must match existing state) FCT
lga LGA name (must match existing LGA) Abuja Municipal
ward Ward name (must match existing ward) City Centre
polling_unit Polling unit name (must match existing PU) PU 001 - Primary School
party Party abbreviation APC
candidate_name Candidate full name John Doe
votes Number of votes (integer) 1234
Tips: State, LGA, Ward, and Polling Unit names must exactly match the entries in the database. Party abbreviations must match registered parties. Duplicate entries (same PU + party) will be updated with the new vote count.
